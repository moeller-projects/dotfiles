# Refactoring Patterns

Safe Local:
- Extract Method
- Rename Method
- Inline Variable
- Replace Temp with Query

Structural:
- Extract Class
- Introduce Interface
- Move Method
- Split Module

Architectural:
- Introduce Boundary Adapter
- Dependency Inversion
- Functional Core / Imperative Shell
- Event-driven extraction

Always:
- Preserve behavior
- Add tests before structural change
